This directory contains example ISL files to generate SVG documents using
xsltd server. SVG capable browser (Firefox, Opera) is required.

Instalation:

- install szarp-xsltd package
- set XSLTD=1 in parcook.cfg
- create subdirectory 'new' in /etc/szarp/default/config
- copy files to 'new' directory
- point your browser to URL like: http://localhost:8084/new/schemat.isl
