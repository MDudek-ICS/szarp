Projekt ISL

Projekt ISL (Industry Schema Language) to projekt, w ramach którego powstaje nowoczesny system oprogramowania wizualizacyjnego dla przemysłu.

Stanowi on część systemu SZARP firmy Praterm, ale może być także dostosowany do współpracy z innymi systemami zbierania i prezentacji danych.

System wykorzystuje nowoczesne technologie, umożliwa łatwy dostęp do danych z poziomu przeglądarki WWW. Do prezentacji danych wykorzystuje SVG (Scalable Vector Graphics) - język do opisu grafiki wektorowej na potrzeby Internetu, opracowany przez organizację World Wide Web Consortium, autora między innymi standardu HTML.

Do oglądania danych prezentowanych przez system można użyc dowolnej przeglądarki WWW, umożliwiającej wyświetlanie grafiki.

Jeżeli przeglądarka nie obsługuje SVG, użytkownik widzi tylko statyczną wersje prezentowanych danych. Aby móc w pełni wykorzystać możliwości systemu, należy korzystać z przeglądarki obsługującej SVG. Do najpopularniejszych przeglądarek można ściągnąć bezpłatną wtyczkę (plugin) firmy Adobe (wersja pod Linuksa dostępna jest jako "old/unsupported"). Pełną obsługe SVG, wraz z możliwościami modyfikacji DOM (Document Object Model), koniecznymi do automatycznego odświeżania wartości parametrów można uzyskać stosując napisaną w Javie przegladarkę Batik.
